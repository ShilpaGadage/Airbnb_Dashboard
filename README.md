Airbnb_Dashboard Project Title / Headline
Global Airbnb Performance Dashboard 
In the Global Airbnb Performance Dashboard, the overall performance from 2008 to 2020 has been analysed across several key dimensions. The dashboard explores trends in listings, cities, hosts, property types, and reviews, helping users understand periods of growth, peak activity, and slowdown in bookings.
It also provides insights into occupancy rates throughout the year for different property types, highlights the most and least popular cities based on reviews, and includes average pricing information for each property type. This allows users to gain a clear understanding of market behaviour and Airbnb performance across regions and years.


Tech Stack
• 📊 Power BI Desktop – Main tool to create a report and data visulisation.
• 📂 Power Query – Data cleaning, data preparing, and  data transformation.
• 🧠 DAX (Data Analysis Expressions) – Created calculated columns and measures to generate dynamic, interactive visuals.
• 📝 Data Modeling – Establishing relationships between the tables, to enable cross filtering and aggregation
• 📁 File Format – .pbix for development and .png for dashboard previews.
